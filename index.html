Getting a WebGL-enabled browser

You need a WebGL-enabled browser to explore these visualizations. Current releases of Chrome and FireFox include WebGL capability and have the feature enabled by default.

WebGL is also included in Safari v5.1.2 but is has to be enabled manually by first enabling the Develop menu in the Advanced tab of the Preferences dialog and then selecting EnableWebGLin the Develop menu.

Learn more about WebGL here: Learning WebGL.
